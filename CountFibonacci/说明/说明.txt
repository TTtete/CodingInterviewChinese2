1、尝试把几个测试函数放在一个指针函数里面。出现一下几个问题没搞明白
（1）函数名代表什么：函数名应该是代表一个指向函数的指针。
证明：
struct Test
{
	void(*test1)(void);
	void(*test2)(void);
	void(*test3)(void);

};
struct Test test = { test1,test2,test3 };
结构体定义的是指向函数的指针变量，而可以用test1，test2等函数名来初始化，
足以证明函数名是指向定义该函数的指针变量。

（2）怎么在指针函数中调用不同的测试函数：在一个函数中挑选不同函数来作用，必须返回一个指向函数的指针类型，
创建返回函数指针类型的函数：void(*ptest(void))(void);  解析：返回函数指针的函数是返回值为void，形参为void的函数。当调用函数的时候，和一般函数一样只要用函数名加（）；――ptest();0
结构体中定义函数指针元素：void(*test1)(void);
定义一个指向函数的指针变量：void(*变量名)(vo);

2、循环用的形参我用register 作为前缀，运算更快。register定义的变量放在寄存器中，可以节省内存空间，加快运算速度。
